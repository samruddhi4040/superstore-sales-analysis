# Superstore Sales Analysis

## 📊 Project Overview
Analyzed retail sales data using SQL and Power BI to identify key business insights related to profitability, discounts, and regional performance.

## 🧹 Data Cleaning
- Handled inconsistent date formats in the dataset
- Converted text-based dates into proper date format for analysis
- Ensured data consistency for accurate reporting and visualization

## 🛠 Tools Used
- SQL (MySQL)
- Power BI

## 🔍 Key Insights
- Despite generating high revenue (~26K), profit was nearly zero, indicating poor margin management.
- Higher discounts (>30%) significantly reduce profitability.
- Furniture category is loss-making compared to Office Supplies and Technology.
- West region contributes the highest profit, while other regions underperform.
- A small number of products drive the majority of revenue.
